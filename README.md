# todo-app-backend

### todo-list api입니다. 
### 아직 완벽히 구현되지 않은  베타버전 입니다.

> 사용하실때에는 db폴더에 있는 secrets.json 파일을 작성하여 사용해주세요.

## Example:
```
{
    "DB":{
        "user": "todo-server",
        "password": "db 비밀번호", # 이부분만 수정해주세요
        "host": "localhost",
        "port": 33306,
        "database": "todo-app"
    }
}
```
